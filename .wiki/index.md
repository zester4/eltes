# Etles Knowledge Wiki

*This wiki is Etles's accumulated expertise. It compounds over time — every great result, every proven framework, every hard-won insight gets compiled here. Agents read the relevant page before starting a task. The wiki grows smarter with every job.*

## How to Use This Wiki

1. Read this index to find the relevant page
2. Load the page with `wikiQuery(action='read', page='...')`
3. Apply the knowledge to the task
4. If you discover something new that should be here, use `wikiIngest` to add it

## Update Rules

- Keep pages dense and specific. No generic advice that any tool could give.
- Every claim should be something that has worked or is a proven framework.
- Add real examples wherever possible.
- Use `wikiIngest` to update a page when you discover something that improves it.
- The wiki is for craft knowledge — how to do great work — not architecture or codebase notes.
